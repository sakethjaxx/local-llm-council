2. Deliberation Mode
Fast
3 independent opinions in parallel. No debate, no cross-examination - quickest answer, weakest guarantee.
Deliberate
Full council: stance gate, cross-review when seats disagree, one rebuttal round, grounded verdict. Slower, trustworthy.
