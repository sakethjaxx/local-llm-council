# Code of Conduct

LLM Council follows the Contributor Covenant Code of Conduct.

## Our Standards

Contributors should keep discussion respectful, specific, and focused on improving the project. Harassment, personal attacks, discriminatory language, and sustained off-topic disruption are not acceptable.

## Enforcement

Project maintainers may remove comments, close issues, or block contributors who violate these standards. If you need to report a concern, contact the maintainer privately or use GitHub's reporting tools.

## Scope

This applies to project spaces including GitHub issues, pull requests, discussions, and community channels linked from the repository.
